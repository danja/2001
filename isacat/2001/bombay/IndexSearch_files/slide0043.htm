Creating Map Reference
start with the cleaned-up name :
page1.html => Page1
if the cleaned-up filename is 'Index‘
use the name of the containing directory instead :
 chapter1/index.html => Chapter1
if this ref has already been used, prepend the last directory name :
chapter1/page1.html => Page1
chapter2/page1.html => Chapter2Page1
e.g.
<mapID target="Chapter1Page1" url="chapter1/page1.html" />
String makeMapRef(String filename, Directory currentDir)
If we have a look at MapDocument’s makeMapRef method, we can see how the filename and directory of the HTML file are ‘humanised’.
