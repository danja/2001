Inference Algorithm
Ø1. Select the unknown fact that occurs as a condition in most rules
Ø
Ø2. Ask the user the state (true/false) of this fact
Ø
Ø3. Modify the rules accordingly - if a goal is proven, go to 7.
Ø
Ø4. Put any newly proven/disproven facts in a queue
Ø
Ø5. Is there a fact in the queue?  If so go to 3.
Ø
Ø6. Are there are any questions left? If so go to 2.
Ø
Ø7. Display results
Ø
The algorithm I’ve used is quite simple, and these are the steps.
