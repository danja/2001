Inference Algorithm
1. Select the unknown fact that occurs as a condition in most rules
2. Ask the user the state (true/false) of this fact
3. Modify the rules accordingly - if a goal is proven, go to 7.
4. Put any newly proven/disproven facts in a queue
5. Is there a fact in the queue?  If so go to 3.
6. Are there are any questions left? If so go to 2.
7. Display results

Puzzle Solution
Only one of the following statements about a particular hand of
cards is true :
1. There is a king in the hand, or an ace or both.
2. There is a queen in the hand, or an ace or both.
3. There is a jack in the hand, or a ten, or both.
Is it possible that there is an ace in the hand?
If there is an ace in the hand, then both 1. and 2. are true, but as only one of
the three statements can be true, then there can't be an ace in the hand.
