MumJava Materials

Here is some of the material from the two presentations I gave at the Wrox Press / Times of India Java conference 'MumJava' in Mumbai (Bombay), March 2001.

Indexing and Searching : Java and Real Information

This presentation gave a little background to indexing and searching, with emphasis on the kind of information found on the web. It introduced two applications, the first of which uses static XML files to index HTML pages, and the second is a prototype search engine.

Building a J2ME Application : A Palm Expert System

This described how an application could be written for the Palm PDA in Java. The application is a simple Expert System, which uses a knowledgebase in XML loaded from a standard Palm Memo.
