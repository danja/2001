Stop words

Lists of common words to filter out of text, used in various information retrieval tasks.

list1 list2 - both plain text, maybe 500 words each

There's some (undocumented) code for a stop filter (and a stemmer) in the IOU stuff. In the utility classes (somewhere) there's the code for building such lists from existing documents.

 

Should really go on the links page, but a few related resources I'd love to get machines to play with are :

The Free On-line Dictionary of Computing, FOLDOC, a (~4.5MB) plain text dictionary, begging to be made properly machine readable

Open Directory (DMOZ) - already machine readable (near RDF), but I've not got around to doing anything with this yet..

WordNet is an *excellent* lexical database (human & machine-readable thesaurus) with viewers for any platform you care to mention (inc. Palm).

What I'd love to see is an open-ended system that could use the 3 data sets above (and any similar ones, including the whole web) and make inferences on them (ermm... that sounds very like the Semantic Web). One day I'll have time for this kind of stuff...
