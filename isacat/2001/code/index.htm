There is some (limited) documentation for AutoHelp, IOU and Kex in the form of presentation materials.

AutoHelp

This is a command-line application which will automatically build the XML HelpSet files needed by JavaHelp. Uses a recursive directory walker to discover files, then indexes them according to filename/directory, uses DOM to build the trees and then writes these to file.
Last Update : 21/02/2001

CSVToXML and XMLToCSV

Lightweight conversion tools - CSV to XML converts text to XML (demonstrating that you don't always need SAX/DOM/XSLT etc) and XML to CSV converts XML to comma-separated text data.
Last Update : 25/02/2001

IOU Webcrawler currently being reworked May 2001 - waiting for new version recommended

This is part of a prototype search engine. It follows links on web pages, scans (parses) the pages and stores key information in a database using JDBC. Includes an config file loader with properties specified in XML.
Last Update : 21/02/2001

KVM Expert System (for the Palm) currently being reworked May 2001 - waiting for new version recommended

This will ask questions based on a knowledge base specified in a Palm Memo (in XML). From the user's responses to the questions (Yes/Know/Don't Know) it will resolve rules in the knowledge base to reach a goal.
Last Update : 21/02/2001

 

Example apps (undocumented) with source (zipped)

DirToXML - builds an XML file from a file system structure (cut-down version of AutoHelp) using DOM

JDBCToXML - converts a ResultSet into XML

XMLToJDBC - uses SAX to put XML data into a RDBMS

DOMFilter - reads in XML (as generated by DirToXML), uses DOM to filter the tree and then outputs the result as an XML file

The above files use the Java API for XML, JAXP (except for JDBCToXML, which doesn't need to), although Xerces can be used with minor modifications (see code comments).

HTMLParser - everything needed to pull a page or file over http and optionally parse it. Uses core Java 2.
Zip contains :

  • HTTPReader.java - reads data from URL, calling parser if required
  • GenericParser.java - interface representing any parser
  • HTMLParser.java - HTML parser
  • EchoHTMLParserCallback.java - example handler, echos tags etc. to console
  • TestHTMLParsing.java - command-line test for the above (gets http://localhost)

Last Update : 25/03/2001
