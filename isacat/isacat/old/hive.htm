Ape* Model for Web Agents

(* Italian for bee pronounced 'ah-pay', allowing additional reference to the Piaggio Ape 3-wheeled workhorse vehicle, from the makers of the Vespa scooter)

Status : an idea

Overview

A hive of bees is a colony of foraging insects, which spend a lot of time out and about gathering nectar from flowers. Each worker bee acts as an individual agent, and when it has successfully found a good source of food it returns to the hive with its payload and communicates to other bees in the hive the location of more nectar by means of the remarkable symbolic waggle dance (also the name of a fine ale). The workers collect two kinds of food - nectar (which gets turned into honey) and pollen.

flowers = web pages (with embedded metadata, or linked RDF)

pollen = RDF

nectar = page that satisfies client

hive = Ape server : consisting of proxy web server, RDF DB, HTML & RDF parsers, web server generating UI page

bee = 1 instance of proxy

UI page contains user feedback frame

 

Client <-> Ape server <-> web

 

Pollenation

In the same way that bees carry pollen from flower to flower thus playing a vital role in the life history of the plant, it is also a possibility that the kind of agent described here could offer a service to hosts it visits - the addition of a few characters to the standard URLs of the pages it visits could facilitate a dialogue between the host and the agent, which could be used to supply the host with select parts of the RDF payload - i.e.

http://www.isacat.net/index.htm (straight http)
http://www.isacat.net/index.htm.ape (call in the ape CGI script/servlet)

Information Foraging - Xerox PARC User Interface Research Group, see also a brief lay article at siliconvalley.com

Swarm - Swarm is a software package for multi-agent simulation of complex systems, originally developed at the Santa Fe Institute. Swarm is intended to be a useful tool for researchers in a variety of disciplines. The basic architecture of Swarm is the simulation of collections of concurrently interacting agents: with this architecture, we can implement a large variety of agent based models. The Swarm software is available to the general public under GNU licensing terms. Swarm is experimental software, which means that it's complete enough to be useful but will always be under development.

Hive is a Java software platform for creating distributed applications. Using Hive, programmers can easily create systems that connect and use data from all over the Internet. At its heart, Hive is an environment for distributed agents to live, communicating and moving to fulfill applications. We are trying to make the Internet alive.
